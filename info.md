# TuyaExtend AmperePoint

AmperePoint Home Assistant helper integration plus EVSE profiles, dashboards,
DP maps and sanitized Tuya observations.

Before installing TuyaExtend AmperePoint, initialize the official Home Assistant
Tuya integration and confirm that the charger is visible in Home Assistant.

After downloading through HACS, restart Home Assistant, add `TuyaExtend
AmperePoint` from Devices & services, and follow the welcome flow. Automatic
setup can create a ready-to-use AmperePoint dashboard in the sidebar without
changing existing dashboards.

See `INSTALL.en.md`, `INSTALL.pl.md`, `README.md` and `amperepoint/README.md`.
